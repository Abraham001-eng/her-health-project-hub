# Her Health Project Hub
Live Site: https://herhealthprojecthub.org.ng/
Responsive landing page for a Nigerian women's health initiative.

## Tech Stack
- HTML5 - Semantic structure
- CSS3 - Flexbox, Grid, Media Queries  
- JavaScript ES6 - Interactive elements

## Features
- 100% Mobile responsive design
- Fast load time & optimized images
- Cross-browser compatible
- Deployed to custom domain
